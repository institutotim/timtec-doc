====================
Documentação TIM Tec
====================

A plataforma `TIM Tec <http://timtec.com.br>`_ é um software livre para criação de cursos online que utiliza o conceito de MOOC (Massive Open Online Courses), cursos abertos, livres que podem ser feitos simultaneamente por muitas pessoas.

`TIM Tec <http://timtec.com.br>`_ é uma iniciativa do **Instituto TIM**!

************************************************
`Manual do Professor <manual_professor.html>`__
************************************************

O sistema pode hospedar múltiplos cursos com videoaulas, além de materiais de apoio e outros links. A ferramenta permite a criação de testes e o uso de emuladores desenvolvidos para que alunos exercitem os conhecimentos abordados.

Alunos podem fazer e responder perguntas, ver respostas e favoritar as dúvidas dos colegas num fórum, além de fazer anotações em seu caderno virtual.

Se você é professor, `veja como utilizar <manual_professor.html>`__.
